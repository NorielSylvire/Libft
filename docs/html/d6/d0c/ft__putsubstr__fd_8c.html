Prints a substring to the designated fd.

Prints a subset of a string to the provided file descriptor. The substring contains all characters from the start position to the end position, both included.

Parameters
strthe string from which to print the substring
startthe index of the first char of the substring
endthe index of the last char of the substring
fdthe file destriptor to print to
Returns
The number of characters that where printed to the fd
