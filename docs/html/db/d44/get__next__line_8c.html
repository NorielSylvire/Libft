static void joinfree ( char ** ret,
char * str,
int chars_read )
static

Executes strnjoin on ret and str, then swaps it with ret.

A new string is created containing the result of joining ret and the chars_read first characters of str. Then the old ret is free'd and ret is assigned this new string.

ret is passed by reference because we need to modify the value of the ret variable in another function, not only in the scope of this function.

Parameters
retthe address of the string that will contain the line read
stra string containing the characters read by the read()

static char * read_line ( int fd,
char * ret )
static

Reads the first line of the file pointed to by fd.

This function takes a file descriptor and attempts to read it in BUFFER_SIZE increments until either an error, the EOF, or a '
' character is encountered. If no errors were encountered, the line that was read is stored in the static buffer ret. At this point, ret may contain extra chars after the '
' which need to be cleaned at a later stage.

If an error is encountered, all buffers are free'd and set to NULL, and NULL is returned.

Parameters
fda file descriptor that points to the file to be read
retthe static buffer in which the line will be stored
Returns
The static line containing the line that was read
